# ToDo App

Простое ToDo-приложение на React + TS + RTK.

## Функционал:
- Добавление задач
- Отметка выполненных
- Фильтрация (все, активные, выполненные)
- Подсчет оставшихся
- Очистка выполненных

## Запуск проекта:
```sh
npm install
npx vite
```

## Деплой:
Приложение доступно по ссылке: [деплой](https://test-mindbox-five.vercel.app/)
